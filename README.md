# 🏡 Airbnb Analytics Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard developed to analyze Airbnb listing data and provide meaningful business insights. The dashboard enables users to explore pricing trends, revenue performance, host analytics, neighborhood comparisons, and property availability through dynamic visualizations and interactive filters.

---

## 🎯 Project Objectives

- Analyze Airbnb listings across different neighborhoods.
- Identify revenue and pricing trends.
- Evaluate host performance.
- Compare average ratings by neighborhood.
- Monitor listing availability.
- Build an interactive dashboard for business decision-making.

## 📊 Dashboard Features

- KPI Cards

  - Average Rating

- Revenue Analysis

- Listings by Neighborhood

- Price Distribution

- Top Hosts by Revenue

- Average Rating Analysis

- Availability Analysis

- Interactive Slicers

- Navigation Panel

## 📈 Key Insights

- Identified neighborhoods generating the highest revenue.
- Compared pricing across different locations.
- Analyzed host performance using revenue metrics.
- Visualized rating trends across neighborhoods.

- Enabled interactive exploration using filters and slicers.
